# Residual Analysis Visualizations

This folder contains bar charts illustrating the distribution of regression residuals used in the performance and valuation analysis.

Each visualization displays the residual values across midfielders in the sample, with a vertical reference line indicating the position of Moisés Caicedo within the distribution.

## Included Visuals

### Defensive Residuals
Bar chart showing the distribution of defensive performance residuals. The horizontal axis represents residual values, while the vertical axis represents frequency.

### Offensive Residuals
Bar chart showing the distribution of offensive performance residuals across the sample.

### Overall Impact Residuals
Bar chart displaying the distribution of aggregate performance residuals constructed from multiple performance dimensions.

### Market Value Residuals
Bar chart showing the distribution of market value residuals, defined as the difference between observed and model-predicted market values.

## Notes on Interpretation

Residuals represent deviations from model-predicted values after controlling for observable player characteristics. The red vertical reference line in each chart denotes Moisés Caicedo’s position relative to the distribution. Interpretation of these visuals is provided in the accompanying report.
